# Geospatial Python Toolkit

A collection of beginner-friendly geospatial Python scripts demonstrating common GIS workflows.

## Tools Used

* Python
* GeoPandas
* Rasterio
* Shapely
* Pandas

## Scripts

### coordinate_converter.py

Convert latitude and longitude coordinates into UTM coordinates.

### buffer_analysis.py

Generate spatial buffers around points for proximity analysis.

### shapefile_reader.py

Read shapefiles and inspect attribute tables and coordinate systems.
